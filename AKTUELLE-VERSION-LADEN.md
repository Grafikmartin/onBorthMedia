# Aktuelle Version im Browser sehen

## 1. Alle Tabs mit der App schließen
- Jeden Tab schließen, in dem `localhost:5173` oder `/onborthmedia/` offen ist.

## 2. Dev-Server neu starten
- In Cursor: **Terminal** öffnen (Ctrl+`).
- In den OnBorthMedia-Ordner wechseln: `cd OnBorthMedia`.
- Falls bereits ein Server läuft: **Ctrl+C** drücken (Stopp).
- Neu starten: `npm run dev`.
- Warten, bis „Local: http://localhost:5173/onborthmedia/“ erscheint.

## 3. Seite ohne Cache laden
- **Neues privates Fenster** oder **Cmd+Shift+R** (Hard Reload).
- Adresszeile: **http://localhost:5173/onborthmedia/** eintippen.

## Wichtig
- Immer **http://localhost:5173/onborthmedia/** nutzen (Port **5173**).
